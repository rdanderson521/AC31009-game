To compile:
First open the game in GoDot
Go to  project and click export in the dropdown
Select the windows desktop preset (you may need to add this with the add button)
Make sure runnable is enabled
Set your build path
In the "options" tab select Embed Pck
In the resources tab set export mode to "Export all resources in the project" 
In"Filters to export non-resource files" enter "*.json" (excluding the "" marks)
Click "Export Project" at the bottom
Untick "export with debug" on the next screen and click save

This will export the .exe which you can run
